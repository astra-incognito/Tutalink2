You are building a full-featured, scalable React-based web platform for student-to-student in-person tutorial bookings. Students can discover and book top-performing peers for physical tutorial sessions on campus.

This project is for a startup MVP and will be expanded later with real-time database and payments. Focus on a responsive, interactive, and animated UI using TailwindCSS and Framer Motion. Use mock data or localStorage where backend isn't implemented yet. The app name is TutaLink 

🔧 Stack:

React (preferably Vite setup)

Tailwind CSS for styling

Framer Motion for page and component animations

react-router-dom for navigation

(Optional: Firebase or localStorage for auth and real-time updates)

User Roles & Authentication

Signup with role selection: Tutor or Learner

Login system using local state or Firebase auth

New users are not active by default — require admin approval

Admin logs in using default credentials:

Username: admin123

Password: password123

Auth state is persistent and role-based (Learner, Tutor, Admin)

Admin Panel

View pending signups and approve/reject users

Dashboard to manage profile, availability, and booking requests

View learner reviews and ratings

Shared Features

Profile page with editable fields (bio, courses, location)

Animated transitions, button interactions, and page loaders

Responsive design for mobile and desktop

Notification system (mocked for now)
